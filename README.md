Selection of powershell scripts to help with the management of windows 11.

--Use windows setup utility for the full UI with various options

--totaldeploy will setup a machine debloating, tweaking and installing applications. primarily used for non-group policy enviroments where you want a quick setup

--quicksanitize debloats and tweaks windows 11

--lockdown is a tool which will disabled internet access for x time and whitelist your chosen domains. Good for an exam enviroment.

--domain migration is a tool in which helps change domains from federated to managed.
